## Versioning Policy

FilterQL follows [Semantic Versioning](https://semver.org/):

- **MAJOR**: Incompatible API changes
- **MINOR**: Backward-compatible new features
- **PATCH**: Backward-compatible bug fixes

### Java Compatibility

| Version | Minimum Java |
|---------|--------------|
| Core 4.x | Java 21+ |
| JPA Adapter 2.x | Java 21+ |
| Spring 4.x | Java 17+ |
